# HackWithInfy
HackWithInfy subwaysufer program

Array program done in CPP in which their are  fixed 3 columns and M number of Rows,
just like subway sufer game where there are three rails tracks and boy run unlimited to M number of rows , 
where their are coins and always boy collect maximum value coin similarly we have to achieve all maximum values 
from the array and add them all and also describe the way.
